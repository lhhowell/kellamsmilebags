Activity Kiosk

A complete reference for deploying the Leadership Activity Kiosk, approving student letters, and printing the class sticker set.

What This Is. Leadership students use this kiosk to create an 8-part package for a child receiving a Smile Bag. Six activities build leadership skills for the recipient. Tab 7 is a personal handwritten letter guided by a Gemini AI tutor and approved by you before printing. Tab 8 is a OneWord sticker you print for the whole class at once.

Before Class
Complete Setup Checklist
Deploy Code_LeadershipLetterReview.gs as a Web App in Google Apps Script — Execute as: Me · Who has access: Anyone.
Add your email address on line 11 of Code.gs before deploying.
Copy the Web App URL and paste it into the kiosk HTML, replacing YOUR_APPS_SCRIPT_URL_HERE.
Set up the Gemini Gem using Leadership_Gemini_Gem_Setup.docx. Copy the Gem link and paste it into the kiosk HTML, replacing GEM_LINK_HERE.
Upload the updated kiosk HTML to GitHub Pages.
Post the kiosk link and the Gemini Gem link on Canvas for students.
Keep Leadership_Sticker_Print_Sheet.html handy — you will use this to print all stickers at once.
Reference
The Eight Tabs — What Each One Produces
Tab 01
🤝 Friendship Cards
Four age-appropriate challenges a child can do with a friend. Student customizes for Elementary, Middle, or High School recipient.
Tab 02
🧩 Puzzle It Out
A riddle connected to the student's OneWord, with an answer and a lesson. Student writes their own version.
Tab 03
💛 Kindness Coupons
Four coupons for genuine acts of service. Student writes the specific acts. Printed and cut.
Tab 04
🌬️ Calm-Down Toolkit
A breathing technique named after the student's word. The interactive breathing circle demonstrates the technique.
Tab 05
✨ Dream Jar
A fill-in activity guiding the child through a dream, a goal, and a message. Age-calibrated prompts.
Tab 06
💡 Big Idea Planner
A structured planning sheet with steps, resources, and purpose. 3–5 steps depending on age group.
Tab 07 · Special
✉️ Letter
Student-written personal letter. Guided by a Gemini tutor. Submitted for your approval. Written by hand on cardstock.
Tab 08 · Special
🏷️ Sticker
2-inch round OneWord sticker designed by the student. Submitted to Google Sheet. You print all stickers as a class set.
Tab 7 · Letter Workflow
Letter Approval — How It Works

This is the most important teacher-facing process in the kiosk. Here is exactly how it works:

1
Student uses the Letter Writing Tutor
The kiosk guides the student through 5 questions one at a time. Their answers are assembled into a draft letter using their own words.
2
Student optionally uses Gemini
The Open in Gemini button opens the approved Gemini Gem in a new tab. Gemini asks the same 5 questions conversationally and helps the student shape their language. Student copies the result back into the kiosk editor.
3
Student submits for review
Student clicks Submit for Teacher Review. The kiosk logs the letter to your Google Sheet and emails you the full letter with a one-click Approve link.
4
Student receives a token code
A short code (like KSB-4F2A) appears on screen. Students write this down. If they close the browser, they can re-enter the code to check approval status when they return — useful across two class periods.
5
You approve by email
Click the Approve link in the email. A confirmation page appears. The student's kiosk unlocks within 20 seconds automatically.
6
Student prints the Writing Guide
The Print Writing Guide button opens a new window with the Gemini draft as a reference at the top and 14 blank handwriting lines below. Student writes their letter by hand on this cardstock sheet.
⚡ Two Class Periods

Tokens never expire. A Period 1 student who doesn't get approved during class can return in Period 2 or the next day, enter their token code on the Letter tab, and check status. The kiosk polls for 15 minutes then stops — students use the token code to resume.

Tab 8 · Sticker Workflow
Sticker Printing — Class Set

Students submit their sticker designs digitally — they do not print individually. You print all stickers as one class set.

  • Open Leadership_Sticker_Print_Sheet.html in Chrome.
  • Paste your Apps Script Web App URL into the URL field.
  • Click Load Stickers — all submissions appear in a 3×4 grid, 12 per page.
  • Load sticker paper into your printer (Avery 22807 or equivalent 2-inch round).
  • Click Print Sheet — one print job, entire class set.
  • Mark the Printed? column Yes in your Google Sheet for each student.
Approval
Quality Indicators — What to Look For
  • Does the letter mention the student's OneWord? If not, ask them to revise.
  • Does it sound like a real person wrote it — or like AI output? The Gemini Gem is configured to use the student's own words, but check anyway.
  • Is it at least 30 words? The kiosk enforces this before submitting, but very short letters may need more.
  • Is the tone appropriate for a child receiving a Smile Bag?
  • If you need a student to revise — simply do not click Approve. Reply to the email with specific feedback. The student can edit in the kiosk and resubmit.
