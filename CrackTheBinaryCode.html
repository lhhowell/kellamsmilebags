🧬 The AP Biology Connection
Binary code and DNA both use a simple two-state system to store information. Binary uses 0 and 1. DNA uses four chemical bases — but they always pair up (A with T, G with C), making it effectively a two-state code. The Human Genome Project decoded 3 billion of these base pairs — that's like cracking a binary message 375 million letters long!
