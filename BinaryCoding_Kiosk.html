The AP Biology Connection
Binary code and DNA both work the same way: two states, infinite combinations. In binary it is 0 and 1. In DNA it is four bases (A, T, G, C) that work in pairs — effectively the same two-state logic. Every living thing on Earth is encoded in a molecular language not so different from the one computers use.
